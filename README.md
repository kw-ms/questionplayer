# このアプリについて
このアプリは問題データからCSVファイルを作成し、シャッフルして閲覧することができるアプリです。わかりやすいナビを使って作成する方法と手動で作成する方法の2つがあります。

# 使用方法
以下のGithub Pagesのサイトから使用するか、ソースコードをダウンロード・展開して使用してください。

> [!NOTE]
> ソースコードから使用する場合は、展開後、index.htmlを任意のブラウザで開いてください。
